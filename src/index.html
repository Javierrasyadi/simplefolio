HalalApp

I built this native Android app to make verifying halal ingredients effortless. Using Kotlin and Google ML Kit, I integrated OCR and Translation APIs. When the initial text scans were inaccurate, I experimented with image preprocessing to successfully boost the accuracy from 60% to over 90%. I also implemented an offline-first architecture using Room and Firebase so it works reliably anywhere.

MedIn

I created MedIn to serve as an accessible mental health triage application. Through a structured questionnaire, the app uses rule-based logic to evaluate a user's current mood and immediately suggest nearby clinics for professional help. Instead of complex machine learning, my main focus here was designing an empathetic, easy-to-navigate UI and writing clean conditional statements to ensure the recommendations felt accurate and supportive.
